# 릴리스 노트 — v0.8.7

《제2차 슈퍼로봇대전》(슈퍼로봇대전 컴플리트 박스 수록)의 화면 내 모든
텍스트를 한국어로 옮긴 릴리스입니다. `v0.2.1-pre` 이후 대사 외의 UI
전 영역을 채우고, 부팅 안전을 유지하면서 이름·메뉴 표기를 정식화했습니다.

## 이번 버전에서 추가/변경

- **시나리오 예고편·프롤로그 컷신 89건 번역.** 부팅 크리티컬한 root VM을
  이동하지 않고 원본 3,235바이트 arena 안에서 대화문을 재구축하도록
  해, 이전 시험에서 발생하던 부팅 멈춤 없이 예고편이 한국어로 나옵니다.
  화자·동적 이름(F9)·페이지 제어(F7)를 보존하고 20칸 창에 맞춰 자동
  줄바꿈합니다.
- **맵 UI 라벨/시스템 메시지 힙(356건) 번역.** 부대표(아군/적군)·반격
  명령·페이즈종료·작전목적·정신검색·유닛개조/무기개조·환승·세이브 및
  메모리카드 안내·지형/지역 이름·강화파츠 설명 등. 참조 방식이 인덱스
  기반인 힙은 레코드 크기와 FF 종결자를 정확히 보존하는 바이트-정확
  in-place 방식으로 교체했습니다.
- **작전 화면 승리/패배 조건 문자열 117건 번역**(2_SCE.BIN, 적 전멸/
  아군 전멸/보스별 파괴·격추 조건 등).
- **시스템 설정·상태·개조 화면 전면 한국어화.** 일어의 반각(가나)·전각
  (한자)과 한글 전각의 폭 차이를 고려해 정확한 위치에 배치합니다.
- **파일럿·유닛 이름 정식 표기.** 표시 컬럼이 고정 폭이라는 점을 반영해,
  과도한 축약(암로·브라잇·료 등) 대신 정식 이름(아무로·브라이트·류마
  등)을 유지하도록 이름 리스트의 폭 상한을 컬럼 폭으로 조정했습니다.
- 페이즈종료·반격명령·작전목적·시스템설정·사운드·강화파츠선택 등
  원본 폭에 들어맞는 메뉴 항목을 정식형으로 되돌렸습니다.

## 검증

- 번역 자산 검사, 전체 구조 검사, 13개 회귀 테스트 통과
- xdelta를 원본 Track 1에 역적용한 결과가 패치 Track 1과 SHA-256 일치
- 부팅 크리티컬 영역(root VM·메모리 맵) 무변경 — DuckStation에서 제2차
  부팅 및 진행 확인
- 변경 표면은 텍스트 테이블·이름 풀·맵 라벨 힙·폰트·재조준 포인터로 한정

## 알려진 사항

- 시나리오 시작 시 대형 타이틀 카드(第2話 등)는 미리 렌더된 그래픽이라
  텍스트 패치 대상이 아닙니다. 시나리오 이름 자체는 로드/세이브 화면
  등에서 한국어로 표시됩니다.
- 정신기 창의 `????`는 원본 게임의 빈(미보유) 슬롯 표시입니다.
- 처음부터 끝까지의 완주 검증은 미완료이며, 일부 화면에서 이름 정렬
  또는 말투 다듬기가 더 필요할 수 있습니다.
