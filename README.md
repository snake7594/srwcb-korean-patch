# 슈퍼로봇대전 컴플리트 박스 한국어 패치

> **v0.9.0 (최신)** — 《제3차 슈퍼로봇대전》 **대사(시나리오·전투)** 한글화를
> 추가했습니다. 하나의 패치로 **제2차(전체) + 제3차(대사)** 가 함께 적용됩니다
> (`srwcb-second-third-korean-v0.9.0-third-dialogue.xdelta`). 제3차 메뉴·이름
> 테이블과 단독판(SRW3) 이식은 추후 업데이트 예정입니다. 자세한 내용은
> [v0.9.0 릴리스 노트](docs/RELEASE_NOTES_v0.9.0.md)를 보세요.

아래 설명은 제2차(`v0.8.7`) 기준이며, 제2차는 v0.9.0에도 그대로 포함됩니다.

`v0.8.7`은 《제2차 슈퍼로봇대전》의 **화면 내 모든 텍스트**를 한국어로
바꾸는 릴리스입니다. 시나리오 대사·전투 대사부터 메뉴·인터미션·시스템
설정·지형/지역 이름·강화파츠 설명·시나리오 예고편 컷신까지 포함합니다.
원문보다 긴 번역을 잘라 넣지 않고, 메시지 저장소와 포인터를 다시 구성하는
가변 길이 패치를 사용하며, 폭이 고정된 이름/메뉴 필드는 렌더러 advance를
그대로 보존합니다.

**두 가지 CD 판을 지원합니다.** 《슈퍼로봇대전 컴플리트 박스》 수록판과,
《제2차 슈퍼로봇대전》을 **단독으로 별매한 CD판**(`SLPS_024.06`) 모두에
같은 번역을 적용할 수 있습니다. 단독판은 실행파일 오프셋이 달라 단독판
기준으로 재빌드했습니다([아래 단독판 섹션](#단독판별매-cd-지원)).

《제3차 슈퍼로봇대전》, 《슈퍼로봇대전 EX》, 《슈퍼로봇대전 외전

그래픽**이라 이번 텍스트 패치 대상이 아닙니다. 시나리오 이름 자체는
로드/세이브 화면 등 모든 텍스트 표시에서 한국어로 나옵니다. 정신기
창의 미보유 슬롯에 표시되는 `????`는 원본 게임의 빈 슬롯 표시입니다.

## 검증 상태

이 버전은 번역 자산 검사, 전체 구조 검사, 13개 회귀 테스트와 xdelta
역적용 검증을 통과했으며, DuckStation에서 **제2차 슈퍼로봇대전 부팅 및
진행이 확인**되었습니다. 다만 처음부터 끝까지의 실기 플레이 완주 검증은
완료되지 않았습니다. 오역, 말투 불일치, 일부 화면 정렬 문제가 남아 있을
수 있습니다. 원본 이미지는 별도로 보관하고 시험용으로 사용하십시오.

## 패치 적용

### 간편 적용 (Windows, 권장)

릴리스에서 easy-apply zip(최신 **`srwcb-cb-korean-v0.9.0-easy-apply.zip`**,
제2차 전용은 `srwcb-second-korean-v0.8.7-easy-apply.zip`)을 받아 압축을 풀면
`xdelta.exe`·패치·적용 스크립트가 모두 들어 있습니다.

1. 정품 디스크에서 추출한 원본 `.bin` 2개(Track 1·Track 2)를 압축을 푼
   폴더에 넣습니다.
2. **`한글패치 적용하기.bat`** 를 더블클릭합니다. (또는 원본 Track 1
   `.bin` 파일을 이 배치파일 아이콘 위로 끌어다 놓아도 됩니다.)
3. SHA-256 검증 → 패치 → 결과 검증이 끝나면 같은 폴더에 한글판 Track 1과
   `.cue` 가 만들어집니다. 에뮬레이터(DuckStation 등)에서 그 **`.cue`** 를
   여세요.

원본 `.bin` 은 변경되지 않으며, 결과물은 자동으로 SHA-256 검증됩니다.
(`.bat` 은 PowerShell 실행 정책을 우회하여 동봉된 스크립트만 실행합니다.)

### 수동 적용

직접 적용하려면 정상 소유한 아래 원본 Track 1과 별도로 구한 xdelta3
실행 파일이 필요합니다. 저장소에는 게임 ROM과 `xdelta.exe`가 포함되지
않습니다(위 간편 적용 zip 에는 `xdelta.exe`가 동봉되어 있습니다).

```text
원본 파일: Super Robot Taisen Complete Box (Track 1).bin
원본 크기: 565,543,104 bytes
원본 SHA-256:
3f25650b588774d55c3bbb5b771779beab408eaca020e9a622133ade323a0f94

패치 결과 크기: 568,386,672 bytes
패치 결과 SHA-256:
862dadb0dbeec279ff0db67701ee1a84c58b39ee0e72a909bc1708a2d32b82f9
```

저장소 루트에 `xdelta.exe`가 있을 때:

```powershell
.\apply_patch.ps1 `
  -SourceTrack1 ".\Super Robot Taisen Complete Box (Track 1).bin" `
  -SourceTrack2 ".\Super Robot Taisen Complete Box (Track 2).bin"
```

`SourceTrack2`는 선택 사항입니다. 지정하면 패치된 Track 1과 원본
Track 2를 연결하는 CUE를 함께 만듭니다. Track 2 자체는 변경하지
않습니다.

직접 적용하려면:

```powershell
.\xdelta.exe -d -s `
  ".\Super Robot Taisen Complete Box (Track 1).bin" `
  ".\release\srwcb-second-korean-v0.8.7.xdelta" `
  ".\Super Robot Taisen Complete Box Second Korean v0.8.7 (Track 1).bin"
```

## 단독판(별매 CD) 지원

《제2차 슈퍼로봇대전》을 **단독으로 별매한 CD판**(부트 `SLPS_024.06`, CloneCD
`.img`)에도 같은 v0.8.7 번역을 이식한 패치를 제공합니다. 같은 게임이지만
실행파일 오프셋이 달라 단독판 기준으로 재빌드했습니다.

```powershell
.\apply_srw2_standalone.ps1 -SourceImg ".\Super Robot Taisen 2.img"
```

→ `Super Robot Taisen 2 (Korean).img`와 `.cue`가 만들어집니다. DuckStation에서
**반드시 생성된 `.cue`를 여세요**(원본 `.ccd`로 열면 "메모리 확인"에서 멈춤).
자세한 내용은 [단독판 릴리스 노트](docs/RELEASE_NOTES_srw2_standalone.md),
이식 도구는 [`standalone/`](standalone/)에 있습니다.

## 가변 길이 패치

한국어 문장을 원래 일본어 바이트 수에 맞추기 위해 화자, 띄어쓰기,
문장 끝을 삭제하지 않습니다.

- 한 줄 20칸, 한 페이지 3줄을 기준으로 줄바꿈과 페이지 전환을 생성
- `2_SCE.BIN`의 시나리오 블록과 B1/B3/B4 참조를 재구성(승리/패배 조건
  문자열 포함)
- `BMESS2.BIN`의 실제 런타임 leaf 대상만 재배치
- `2_DEAD.BIN`의 start/end 슬롯을 재구성
- 실행 파일의 메뉴 스크립트·문자열 테이블·이름 테이블을 실행 파일 내부
  정적 여유 영역으로 재배치하고 모든 포인터를 갱신
- 시나리오 예고편 풀은 부팅 크리티컬한 root VM을 옮기지 않고, 원본 3,235
  바이트 arena 안에서 재구축하며 참조 포인터만 재조준
- 맵 UI 라벨·시스템 메시지 힙은 레코드 크기와 FF 종결자를 정확히
  보존하는 바이트-정확 in-place 방식으로 교체
- 파일럿·유닛 이름 리스트는 고정 표시 컬럼 폭 안에서 정식 이름을 유지
- 커진 파일을 새 ISO extent로 옮기고 양방향 디렉터리 필드를 갱신
- 변경한 MODE2 Form 1 섹터의 EDC/ECC 재계산

세부 구조는 [확장 빌드 문서](docs/SECOND_EXPANDED_BUILD.md),
[시나리오 재배치](docs/SECOND_SCE_RELOCATION.md),
[전투·격추 메시지 저장소](docs/SECOND_MESSAGE_ARCHIVES.md)에서 확인할 수
있습니다.
