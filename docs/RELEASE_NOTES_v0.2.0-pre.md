# v0.2.0-pre — 제2차 슈퍼로봇대전 메뉴·인터미션 한국어 패치

## 포함 내용

- 시나리오·전투·격추 대사 22,075개 적용 위치, 한국어 번역 6,503종
- 프롤로그·조건·선택지 UI 스크립트 91건
- 음악·데모 제목 171건
- 무기·정신기·강화 파츠·지형·능력·시나리오 제목 1,754건
- 파일럿·기체 이름 1,248건
- 제2차 전용 메뉴·인터미션 레코드 73건
- 제2차 실행 경로의 공통 메뉴 레코드 5건

《제2차 슈퍼로봇대전》 실행 경로에서 조사된 메뉴·인터미션·공통 UI를
한국어로 바꿉니다. 합본에 수록된 다른 세 게임의 대사와 각 게임 전용
메뉴는 이번 릴리스 범위가 아닙니다.

## 구조 변경

메뉴 문자열을 원래 슬롯 크기에 맞춰 자르지 않습니다. `SECOND.WAR`의
루트 VM 스크립트, 프롤로그 풀, 음악·데모 제목, 문자열 테이블과 이름
테이블을 확장 arena로 옮기고 참조 포인터를 갱신했습니다. 커진
`SECOND.WAR`도 메시지 저장소 세 파일과 함께 새 ISO extent로 옮깁니다.

전투 화자 이름용 4×512바이트 scratch는 BSS에 따로 예약했습니다. BSS
초기화 경계와 BIOS heap 시작을 확장 실행 이미지 뒤로 옮겨, 화자 사진
깨짐이나 전투 진입 정지의 원인이었던 메모리 충돌을 피합니다.

공통 음향 옵션은 한국어 `모노`/`스테레오`가 들어갈 수 있도록 두 호출
지점의 선택지 폭을 4바이트에서 8바이트로 늘렸습니다.

## 번역·구조 검증

- 메뉴 스크립트 335레코드, 708개 치환: 일본어·한자 잔존 0
- 메뉴 테이블 8개, 1,754/1,754 적용
- 이름 테이블 3개, 1,248/1,248 적용
- source guard, 포인터 대상, span 겹침, 제어 바이트 오류 0
- 전체 파이프라인 회귀 테스트 11/11 통과
- 전투 scratch 최대 사용량 366/512바이트
- 최종 heap 583,676바이트, 최소 기준 393,216바이트 통과
- 네 ISO extent의 내용·LBA·크기와 MODE2 Form 1 EDC/ECC 검증 통과
- xdelta VCDIFF appheader 비활성화 및 로컬 경로 미포함 검사 통과
- xdelta 독립 decode 결과가 최종 Track 1과 바이트 단위로 일치

빌드 과정에서 에뮬레이터는 자동 실행하지 않았습니다.

## 알려진 제한

- 처음부터 끝까지의 전체 플레이스루 검증은 완료되지 않았습니다.
- 합본의 다른 세 게임 대사와 각 게임 전용 메뉴는 번역하지 않습니다.
- 특정 분기·전투 조합에서 아직 발견되지 않은 문제가 남아 있을 수
  있으므로 원본 이미지를 덮어쓰지 마십시오.

게임 ROM, 추출 실행 파일·데이터, 일본어 원문 원장, BIOS, 에뮬레이터,
`xdelta.exe`는 저장소와 릴리스에 포함하지 않습니다. Track 2는 변경하지
않습니다.
