# v0.9.0 — 제3차 슈퍼로봇대전 대사 추가 (컴플리트 박스)

**《슈퍼로봇대전 컴플리트 박스》 한 장에 제2차(전체) + 제3차(대사)를 함께 한글화**한
릴리스입니다. 하나의 xdelta 패치로 두 게임이 함께 적용됩니다.

## 새로 추가된 것 — 제3차 슈퍼로봇대전 (대사)

- **시나리오·전투 대사 전면 한글화**: 고유 번역 8,639종 / 전체 적용 21,244건
  - `3_SCE.BIN`(시나리오) · `BMESS3.BIN`(전투) · `3_DEAD.BIN`(격추/사망)
  - 제2차와 공유되는 대사 5,017종 재사용 + 제3차 신규 3,622종 번역
  - 화자·동적 이름·페이지 제어 보존, 원문보다 긴 번역을 잘라내지 않는 가변 길이 방식
- **THIRD.WAR 런타임 패치**: 한글 폰트·추가 글리프 주입, 임베디드 BMESS3 테이블
  교체(첫 전투 정지 방지), 한글 전투 텍스트용 배틀 스크래치 재배치(BSS/heap 경계 이동)
- 커진 데이터 파일은 새 ISO extent로 옮기고 디렉터리·EDC/ECC를 재구성

제2차는 v0.8.7과 동일하게 **화면 내 모든 텍스트(대사·메뉴·이름·시스템)** 가 한글입니다.

## 적용 방법

### 🖱️ 간편 적용 (Windows, 권장)

**`srwcb-cb-korean-v0.9.0-easy-apply.zip`** 를 받아 압축을 풀면 `xdelta.exe`·패치·적용기가
모두 들어 있습니다.

1. 정품에서 추출한 원본 `.bin` 2개(Track 1·Track 2)를 압축 푼 폴더에 넣습니다.
2. **`한글패치 적용하기.bat`** 더블클릭 (또는 원본 Track 1 `.bin` 을 배치파일 위로 드래그).
3. SHA-256 검증 → 패치 → 결과 검증 후, 같은 폴더에 한글판 `.bin` 과 **`.cue`** 가 생깁니다.
   에뮬레이터(DuckStation 등)에서 그 `.cue` 를 여세요.

### 수동 적용

```
xdelta.exe -d -s "Super Robot Taisen Complete Box (Track 1).bin" ^
  srwcb-second-third-korean-v0.9.0-third-dialogue.xdelta ^
  "Super Robot Taisen Complete Box Korean v0.9.0 (Track 1).bin"
```

| 파일 | 크기 | SHA-256 |
|------|------|---------|
| 원본 Track 1 | 565,543,104 | `3f25650b…0f94` |
| 패치 결과 Track 1 | 570,075,408 | `e0ed37aa…cf90` |

## 아직 남은 것 (추후 업데이트)

- **제3차 메뉴·이름 테이블**(파일럿·기체·무장·시나리오명 등)은 이번 릴리스에 미포함 —
  일부 메뉴/이름은 아직 일본어로 보일 수 있습니다.
- **제3차 단독판(별매 CD, SRW3)** 이식은 추후. (제2차 단독판은 이 릴리스에도 포함:
  `srw2-standalone-korean-v0.8.7.xdelta` + `apply_srw2_standalone.ps1`)
