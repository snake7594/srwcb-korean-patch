# v0.9.3 — 종료(전원 끄기) 메시지 한글화

타이틀 화면에서 게임을 **종료할 때 나오는 캐릭터 대사**(「그럼, 전원을 끄면
OK다」 류)가 그동안 일본어로 남아 있었습니다. 이번 버전에서 **제2차·제3차의
종료 메시지를 전부 찾아 한국어로** 옮겼습니다.

## 무엇을 고쳤나

종료 메시지는 대사·메뉴와 다른 **타이틀 화면 VM**이 고정 오프셋으로 참조하는
별도 풀에 들어 있어 이전 패치 범위에서 빠져 있었습니다. VM의 메시지 포인터
(`b3` + 필드 상대 16비트 오프셋)를 역분석해 각 레코드의 정확한 위치를 확인하고,
**원본 바이트 예산 안에서 제자리(in-place) 교체**했습니다(포인터 재조정 없음).

- **제2차: 7개 레코드** — 마사키·슈우·비앙·코우지·료마
- **제3차: 14개 레코드** — 아기하·비가지(2)·웬드로(3)·코우지·료마·표마·반죠

컴플리트 박스 디스크에는 `SECOND.WAR`가 **게임 영역과 부트 영역 두 벌** 들어
있어(종료 화면이 어느 쪽을 읽어도 되도록) **두 벌 모두** 교체했습니다.
주입 후 28개(제2차 7+7, 제3차 14) 레코드를 디스크에서 다시 디코드해
**의도한 한국어와 바이트 단위로 일치**함을 확인했고, 세 이미지 모두 xdelta
역적용(retail→패치 결과) 재현을 통과했습니다.

## 적용

세 판 모두 **간편 적용 zip**(더블클릭)과 **xdelta**(수동), **별도 `.cue`** 를
제공합니다. 적용법은 v0.9.2와 동일합니다.

| 판 | 간편 적용 zip | 패치(xdelta) | .cue (별도) |
|----|---------------|--------------|-------------|
| **컴플리트 박스** (제2차 전체 + 제3차 전체) | `srwcb-cb-korean-v0.9.3-easy-apply.zip` | `srwcb-second-third-korean-v0.9.3.xdelta` | `srwcb-cb-korean-v0.9.3.cue` |
| **제3차 단독판** (SLPS-02530) | `srw3-standalone-korean-v0.9.3-easy-apply.zip` | `srw3-standalone-korean-v0.9.3.xdelta` | `srw3-standalone-korean-v0.9.3.cue` |
| **제2차 단독판** (SLPS-02406) | `srw2-standalone-korean-v0.9.3-easy-apply.zip` | `srw2-standalone-korean-v0.9.3.xdelta` | `srw2-standalone-korean-v0.9.3.cue` |

zip 을 풀고 원본을 넣은 뒤 **`한글패치 적용하기.bat`** 더블클릭하면 한글판
이미지와 `.cue` 가 자동으로 만들어집니다. 수동 적용(xdelta 직접)하시는 분은
위 별도 `.cue` 를 받아 패치된 이미지와 같은 폴더에 두고 여세요.

| 파일 | SHA-256 |
|------|---------|
| 패치 결과 CB Track 1 | `be073775…be3b` |
| 패치 결과 제3차 단독판 | `81749469…1968` |
| 패치 결과 제2차 단독판 | `1a14d915…2032` |

전체 자산 해시는 [`release/SHA256SUMS_v0.9.3.txt`](../release/SHA256SUMS_v0.9.3.txt)
에 있습니다.
