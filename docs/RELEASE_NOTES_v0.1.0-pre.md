# v0.1.0-pre — 제2차 슈퍼로봇대전 한국어 패치 조기 시험판

## 포함 내용

- 《제2차 슈퍼로봇대전》 시나리오 대사 4,616건
- 전투 메시지 17,364건
- 격추·사망 대사 95건
- 전체 22,075개 적용 위치, 중복을 합친 한국어 번역 6,503종
- 갈무리14 기반 한글 2,350자와 빌드에 필요한 추가 글리프
- 한 줄 26칸, 한 페이지 3줄 기준의 자동 줄바꿈·페이지 나눔
- 가변 길이 메시지 저장소 재구성과 ISO extent 재배치
- MODE2 Form 1 EDC/ECC 재계산
- 원본·xdelta·출력 SHA-256을 확인하는 PowerShell 적용 스크립트

## 이전 시험판과의 차이

이 버전은 글꼴 표시 시험을 넘어 실제 한국어 대사를 넣습니다. 원래
슬롯 길이에 맞추기 위해 화자명, 띄어쓰기, 문장부호 또는 문장 끝을
삭제하던 고정 길이 번역은 전부 폐기했습니다.

`2_SCE.BIN`, `BMESS2.BIN`, `2_DEAD.BIN`을 각각 구조에 맞게 다시
만들고, 커진 파일을 Track 1의 새 영역으로 옮깁니다. 전투 메시지는
단순 문자열 검색 결과가 아니라 실행 중 leaf가 실제로 참조하는
레코드만 바꿉니다.

## 검증 결과

- 번역 overlay: 6,503/6,503 키 존재, partial=false
- 적용 대상: 22,075/22,075건
- 정적 번역 일관성 검사: 오류 0건, 경고 0건
- 가변 길이 파이프라인 회귀 테스트: 8개 통과
- SCE 직접 참조와 BMESS/DEAD 포인터 구조 재파싱 통과
- BMESS 런타임 슬롯 최대 `0x3000`바이트 제한 검사 통과
- 새 ISO extent의 파일 내용과 디렉터리 LBA·크기 검사 통과
- 변경 섹터 EDC/ECC 재계산 검사 통과
- xdelta 생성 후 독립 decode 결과가 최종 Track 1과 바이트 단위로 일치

## 알려진 제한과 위험

- 번역 6,503종 전체에 대한 별도 검수자의 독립 인적 교정은 완료되지
  않았습니다. 현재 번역 overlay의 항목은 `translated_unreviewed`
  상태입니다.
- 전체 게임을 처음부터 끝까지 진행하는 플레이스루 검증은 하지
  않았습니다.
- 실행 파일에 있는 공통 메뉴와 일부 UI는 번역하지 않았습니다.
- 합본의 다른 세 게임 대사는 번역하지 않았습니다.
- 특정 분기, 전투 조합 또는 에뮬레이터에서 아직 발견되지 않은 문제가
  있을 수 있습니다.

이 버전은 위 제한을 감수하고 결과를 조기에 확인하기 위한
프리릴리스입니다. 원본 이미지를 덮어쓰지 마십시오.

## 배포에서 제외한 자료

게임 ROM, 추출된 실행 파일과 데이터 파일, 일본어 원문 대사 원장,
BIOS, 에뮬레이터, xdelta 실행 파일은 포함하지 않습니다. 저장소에는
자체 제작 도구, 한국어 번역 overlay, 용어집, 기술 문서와 글꼴
라이선스만 제공합니다.

Track 2는 변경하지 않습니다.
